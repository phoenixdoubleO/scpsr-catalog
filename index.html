Setup

1. Go to GitHub → Profile → Settings
2. Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token with repo scope
4. Copy the ghp_... token
5. Log into this panel → paste token when prompted

The token is only held in memory for your session. It is never written to the file or committed to GitHub.
